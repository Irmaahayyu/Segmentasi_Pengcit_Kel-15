# Tugas Segmentasi Pengolahan Citra

Repository ini berisi:

- 4 citra untuk proses segmentasi:
  - original.jpg
  - grayscale.jpg
  - saltpepper.jpg
  - gaussian.jpg
- File kode Python:
  - segmentasi.py
- Folder hasil_segmentasi (berisi output edging untuk Roberts, Prewitt, Sobel, dan Frei-Chen)

## Cara Menjalankan
1. Install library berikut:
   pip install scikit-image opencv-python pillow matplotlib scipy

2. Jalankan kode:
   python segmentasi.py

3. Hasil segmentasi akan muncul otomatis di folder:
   hasil_segmentasi/
